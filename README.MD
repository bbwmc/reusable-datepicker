# reusable-datepicker
A Reusable datepicker for internal usage

Requires:

- jQuery
- momentjs
- flatpickr

id WITHOUT the #

```
STC.Components.BookDatePicker(id, options)
```

See: https://chmln.github.io/flatpickr/options/ for details for options

```
<script>
  $(function() {
    STC.Components.BookDatePicker('date_range')
  });
</script>
```
